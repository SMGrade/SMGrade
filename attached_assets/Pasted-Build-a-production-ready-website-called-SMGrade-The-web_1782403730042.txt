Build a production-ready website called SMGrade.

The website is an AI-powered SwordMasters account grader.

Before generating any code, thoroughly analyze the attached text file. It contains hundreds of real SwordMasters bot command logs from players, along with gear information. Learn from this dataset first.

Do not ignore the uploaded file or make assumptions if the data already provides the answer.

From the dataset, build an internal benchmark database containing:

Average power by level range.
Strong, average, weak, and elite players.
Sword progression.
Shield progression.
Gear quality.
Gold progression.
Overall account progression.

The website must allow users to:

Paste a SwordMasters bot command exactly as copied from Discord.
Upload a screenshot of the bot command.

Automatically extract all player information without requiring users to edit the text.

Extract:

Username
Level
Power
Gold
Sword
Sword Level
Shield
Shield Level
Any additional useful statistics.

The website must automatically remove Discord formatting, emojis, icons, markdown, and unnecessary characters before parsing the data.

The player's score must NOT be generated by AI.

Instead, build a deterministic scoring engine using the uploaded dataset.

Score the account based on:

Sword quality
Shield quality
Power compared to players of a similar level
Gear progression
Gold progression
Overall progression

Gear Score

Strengths
Weaknesses
